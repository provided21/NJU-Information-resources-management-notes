## 说明

这份复习笔记也是我在期末前 4 天左右开始，跟着逐章 PPT 一边看一边整理出来的，主要用于加深自己的理解和记忆，不代表真实考试重点。

这门课建议尽量争取把每个部分都搞懂。考试考得比较细，而且题量不小，我当时都没有完全做完，所以简答题不要写得太铺开，抓住要点答就好。

最后老师好像主要是说了哪些内容不考，没有给出特别确切的重点。因此复习时更适合边看边记，遇到细节及时整理，靠理解把框架搭起来。
