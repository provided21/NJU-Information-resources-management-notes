# NJU Information Resources Management Notes

南京大学信息资源管理学院课程笔记与考试资料整理。

## 课程目录

| 课程 | 内容 |
|---|---|
| [信息组织](./信息组织/) | 章节笔记、2025 回忆版考试资料 |
| [信息检索](./信息检索/) | 2025 回忆版考试资料 |
| [数据科学与数据分析](./数据科学与数据分析/) | 回忆版试卷、复习笔记 |

## 维护方式

- 按课程名建立一级目录。
- 每门课程目录下保留一个 `README.md`，说明课程资料范围与文件索引。
- 章节笔记统一放入 `章节笔记/`，考试资料统一放入 `考试资料/`。
