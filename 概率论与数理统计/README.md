## 说明

这门课我是考前两天才开始复习（更准确说像是预习）。第一天先在 B 站看了一遍框框老师的课程，第二天把整个学期的作业全部自己重新做了一遍（之前很多作业基本都是 AI 做的），然后就去考试了。

这份笔记主要来自老师的复习 PPT、B 站网课笔记，以及我重做作业题时整理出的理解。
