## 说明

这份复习笔记是我在考试前 4 天才开始系统学习这门课时，根据逐章 PPT 边看边整理的。它更像是为了加深自己记忆和理解的学习痕迹，不一定等同于严格意义上的考试重点。

不过从实际考试来看，老师最后画的重点还是比较准的，所以大家不用太焦虑。提前 10 天左右开始认真学这门课也完全来得及，核心还是注重理解，把概念、方法适用场景、基础计算和函数输出看懂。有些实在没法硬记的内容，就适当放过它，别被细枝末节拖住。
